# Tournament Management – Admin Flow (MVP)

This document describes the **admin workflow** for managing a tournament in the MVP version of the system.

The focus of the MVP is **simplicity and speed**, allowing the admin to manually control most operations.

---

# Overview

Admin manages the entire tournament lifecycle:

1. Create tournament
2. Open registration
3. Manage registrations
4. Close registration
5. Create groups (standings)
6. Create match schedule
7. Enter match results

Because this is an **MVP**, several steps are **manual** to reduce algorithm complexity.
